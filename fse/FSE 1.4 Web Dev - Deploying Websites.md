To get Pages to appear, I needed to mod some settings in my jacobsternx for publishing, create a docs/ folder, and since I don't want to install Ruby Gems to be able to use Jekyl templates, I also created a .nojekyl file in the docs folder. Pages is new for me!
Also, since codecademy has a separate repo, I looked into having multiple open folders in VS Code. The best way to do this seems to be to open multiple VS Code windows, although multiple folders can also be added to a workspace through the VS Code file menu.
Also, it may be possible to use GitHub Pages with repos other than username:
 GitHub Pages are published at the URL: YOUR_USERNAME.github.io. So if this link takes your username, what happens if you have multiple repos under your GitHub username? Won't these conflict? Well, GitHub solves this by publishing subsequent repos at the URL: YOUR_USERNAME.github.io/YOUR_REPO_NAME
Also: https://dev.to/github/how-to-use-github-pages-to-host-your-website-even-with-multiple-repos-27k2
